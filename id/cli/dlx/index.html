Version: 6.x

pnpm dlx

Ditambahkan dalam: v6.13.0

Mengambil paket dari registri tanpa menginstalnya sebagai dependensi, meng-hotloadnya, dan menjalankan perintah biner apapun yang dieksposnya.

For example, to use create-react-app anywhere to bootstrap a react app without needing to install it under another project, you can run:

pnpm dlx create-react-app ./my-app

Ini akan mengambil create-react-app dari registri dan menjalankannya dengan argumen yang diberikan.

Since v6.23.1 you may also specify which exact version of the package you'd like to use:

pnpm dlx create-react-app@next ./my-app

--paket <nama>‚Äã

Paket yang perlu diinstal sebelum menjalankan perintah.

Example:

pnpm --package=@pnpm/meta-updater dlx meta-updater --help
pnpm --package=@pnpm/meta-updater@0 dlx meta-updater --help

Pada v6.14.5, beberapa paket dapat disediakan untuk instalasi:

pnpm --package=yo --package=generator-webapp dlx yo webapp --skip-install
